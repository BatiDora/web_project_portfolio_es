# Portafolio RAVA

## Descripción

Portafolio web personal desarrollado para presentar mi perfil como desarrollador Full-Stack Junior, mostrando información sobre mí, mis habilidades técnicas y algunos de mis proyectos más destacados.

El sitio cuenta con un diseño moderno, responsivo y enfocado en ofrecer una experiencia de usuario clara e intuitiva. Además, incluye enlaces para descargar mi currículum, acceder a mi perfil de GitHub y contactar conmigo mediante diferentes redes sociales.

## Funcionalidades

- Presentación personal.
- Sección de habilidades y herramientas utilizadas.
- Visualización de proyectos destacados.
- Enlaces a demostraciones en vivo y repositorios de GitHub.
- Acceso para descargar el currículum.
- Sección de contacto con correo electrónico y redes sociales.
- Diseño adaptable para computadoras, tablets y dispositivos móviles.

## Tecnologías y técnicas utilizadas

### Tecnologías

- HTML5
- CSS3
- Git
- GitHub
- GitHub Pages

### Técnicas de desarrollo

- Metodología BEM para la organización de clases CSS.
- Flexbox para la distribución de elementos.
- CSS Grid para la organización de tarjetas de proyectos.
- Media Queries para diseño responsivo.
- Uso de `clamp()` para tipografía y espaciados adaptables.
- Uso de `aspect-ratio` para mantener proporciones consistentes en las imágenes.
- Pseudoelementos (`::before` y `::after`) para efectos visuales.
- Gradientes radiales y lineales mediante `linear-gradient()` y `radial-gradient()`.
- Transiciones y animaciones CSS para mejorar la experiencia de usuario.
